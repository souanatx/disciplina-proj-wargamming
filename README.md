Welcome to the "Projeto - Wargamming" exercises repository!

<h2>Overview</h2>
    This repository contains all the exercises completed during a class oriented to do a project using all the knownledge acquired through the course, which is part of the Network Administration and Cybersecurity course at CESAE (Centro para Desenvolvimento das CompetÃªncias Digitais).
    The course was in Portuguese so the content of this repository will be aswell.

<h2>About the Course</h2> 
    This class focused on various topics such as strategic and operational impact of cyber threats, operations in computer networks, identification of critical data for organizations, the attack chain (KillChain), articulation between defense and attack, knowledge of the organization's networks, defense in depth, definition of metrics, development of cybersecurity scenarios and framework for the use of simulation exercises (CTFs and "Red and Blue").

<h2>Contents</h2>  
    In this repository, you will find a collection of exercises completed during the course. Each exercise is stored in its respective folder, organized according to the topics covered in the class. Additionally, you can find relevant code examples, projects, and any additional resources that might be helpful for further study.
